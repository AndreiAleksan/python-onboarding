# Description

This repository was made to store a rudimentary "word counter". However, not only it reads words, 
but characters and lines as well. 

## How to run the project:

The program will ask for the path of the file. You must insert it (without quotation marks!). The
output received will be the amount of words, characters and lines inside the file.
